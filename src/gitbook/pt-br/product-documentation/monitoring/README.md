# Monitoramento

Em nossa seção de Monitoramento, você poderá criar armazéns e _pipelines_ de eventos relacionados a todos os nossos produtos e às suas campanhas atualmente ativas. Isso fornecerá dados valiosos para que você estude melhores soluções para suas campanhas e aprimore sua estratégia.

## Armazenamento de Eventos

É possível criar inúmeros armazenamentos de eventos para armazenar dados e definir suas próprias configurações de retenção de dados, permitindo que você ajuste a retenção de acordo com seu planejamento. [Saiba mais sobre Armazenamento de Eventos](./#event-stores).

## Pipelines de Eventos <a href="#event-pipes" id="event-pipes"></a>

Aqui você decidirá de quais eventos coletará dados, abrangendo desde uma campanha ativa até qualquer evento que ocorra em nossa plataforma. Isso fornece amplos dados para ajudá-lo a acompanhar qualquer produto importante no qual você esteja focado. [Saiba mais sobre Pipelines de Eventos](./#event-pipes).
